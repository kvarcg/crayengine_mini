Detailed Description

Functions for the generation of shapes. Both parametric and polygon group shapes are supported.

Creates an axis-aligned rectangle primitive on the XY axis.

Parameters
min_xThe minimum value in the X axis of the rectangle.

min_yThe minimum value in the Y axis of the rectangle.

offsetThe offset of the rectangle in the Z axis.
flip_normalsA flag that dictates if the primitive's normals are flipped.

material_idThe material id that will be used by the primitive.
