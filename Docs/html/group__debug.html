end_yThe end pixel y value to stop logging (Default: none).

void CR_debug_updateCameraPerspective ( cr_vec3  position,
cr_vec3  target,
cr_vec3  up,
cr_float  aperture,
cr_float  near,
cr_float  far 
)

Debug utility that updates camera parameters for navigation during the interactive mode. This is only useful as part of a GUI.

Parameters
positionThe position of the camera in world units.
targetThe target of the camera in world units.
upThe up vector of the camera. This does not need to be accurate as it gets recalculated internally.
apertureThe vertical FOV of the camera.
nearThe near field of the camera. This is not necessary, but is provided as an extra parameter.
farThe far field of the camera. This is not necessary, but is provided as an extra parameter.
