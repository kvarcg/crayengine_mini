Associates an area light with a primitive. Note that the same light source can be shared by multiple primitives. In that case, the light\textquotesingle{}s flux is scaled by each primitive\textquotesingle{}s area. This way, it is easy to associate a large polygon group with one light source. Since this method is approximate, accurate rendering requires 1 light per primitive.
